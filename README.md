# Calcul-DL-california

Application Streamlit simulant un permis de conduire Californien avec génération de code-barres PDF417.

## Structure
- `driver_license_final_fixed.py` : application principale Streamlit
- `pdf417gen/` : module vendorisé pour générer le code-barres PDF417
- `requirements.txt` : dépendances (Streamlit)
- `.gitignore` : fichiers ignorés
